Adds a SkeletonRenderSeparator and child SkeletonPartsRenderer GameObjects to a given SkeletonRenderer.

반환값
The to skeleton renderer.
매개변수
skeletonRendererThe target SkeletonRenderer or SkeletonAnimation.
sortingLayerIDSorting layer to be used for the parts renderers.
extraPartsRenderersNumber of additional SkeletonPartsRenderers on top of the ones determined by counting the number of separator slots.
sortingOrderIncrementThe integer to increment the sorting order per SkeletonPartsRenderer to separate them.
baseSortingOrderThe sorting order value of the first SkeletonPartsRenderer.
addMinimumPartsRenderersIf set to true, a minimum number of SkeletonPartsRenderer GameObjects (determined by separatorSlots.Count + 1) will be added.
