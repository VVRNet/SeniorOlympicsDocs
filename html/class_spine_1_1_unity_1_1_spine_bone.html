includeNoneIf true, the dropdown list will include a "none" option which stored as an empty string.
fallbackToTextFieldIf true, and an animation list source can't be found, the field will fall back to a normal text field. If false, it will show an error.
dataFieldIf specified, a locally scoped field with the name supplied by in dataField will be used to fill the popup results.
