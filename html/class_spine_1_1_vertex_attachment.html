virtual void Spine.VertexAttachment.ComputeWorldVertices ( Slot slot,
int start,
int count,
float[] worldVertices,
int offset,
int stride = 2 )
virtual

Transforms the attachment's local Vertices to world coordinates. If the slot's Slot.Deform is not empty, it is used to deform the vertices.

See World transforms in the Spine Runtimes Guide.

매개변수
startThe index of the first Vertices value to transform. Each vertex has 2 values, x and y.
countThe number of world vertex values to output. Must be less than or equal to WorldVerticesLength - start.
worldVerticesThe output world vertices. Must have a length greater than or equal to offset + count .
offsetThe worldVertices index to begin writing values.
strideThe number of worldVertices entries between the value pairs written.
