Smart popup menu for \doxylink{namespace_spine}{Spine} Attachments 


\begin{DoxyParams}{매개변수}
{\em current\+Skin\+Only} & Filters popup results to only include the current \doxylink{class_spine_1_1_skin}{Skin}. Only valid when a \doxylink{class_spine_1_1_unity_1_1_skeleton_renderer}{Skeleton\+Renderer} is the data source.\\
\hline
{\em return\+Attachment\+Path} & Returns a fully qualified path for an \doxylink{class_spine_1_1_attachment}{Attachment} in the format "{}\+Skin/\+Slot/\+Attachment\+Name"{}. This path format is only used by the \doxylink{class_spine_1_1_unity_1_1_spine_attachment}{Spine\+Attachment} helper methods like \doxylink{class_spine_1_1_unity_1_1_spine_attachment_a853967ef5a05a6e3e4fd4f949eae272c}{Spine\+Attachment.\+Get\+Attachment} and .Get\+Hierarchy. Do not use full path anywhere else in \doxylink{namespace_spine}{Spine}\textquotesingle{}s system.\\
\hline
{\em placeholders\+Only} & Filters popup results to exclude attachments that are not children of \doxylink{class_spine_1_1_skin}{Skin} Placeholders\\
\hline
{\em slot\+Field} & If specified, a locally scoped field with the name supplied by in slot\+Field will be used to limit the popup results to children of a named slot\\
\hline
{\em skin\+Field} & If specified, a locally scoped field with the name supplied by in skin\+Field will be used to limit the popup results to entries of the named skin\\
\hline
{\em include\+None} & If true, the dropdown list will include a "{}none"{} option which stored as an empty string.\\
\hline
{\em fallback\+To\+Text\+Field} & If true, and an animation list source can\textquotesingle{}t be found, the field will fall back to a normal text field. If false, it will show an error.\\
\hline
{\em data\+Field} & If specified, a locally scoped field with the name supplied by in data\+Field will be used to fill the popup results. Valid types are \doxylink{class_spine_1_1_unity_1_1_skeleton_data_asset}{Skeleton\+Data\+Asset} and \doxylink{class_spine_1_1_unity_1_1_skeleton_renderer}{Skeleton\+Renderer} (and derivatives) If left empty and the script the attribute is applied to is derived from Component, Get\+Component$<$\+Skeleton\+Renderer$>$() will be called as a fallback. \\
\hline
\end{DoxyParams}
