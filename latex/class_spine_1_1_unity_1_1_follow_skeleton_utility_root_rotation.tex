Note that flipping needs to be performed by 180 degree rotation at \doxylink{class_spine_1_1_unity_1_1_skeleton_utility}{Skeleton\+Utility}, by setting \doxylink{class_spine_1_1_unity_1_1_skeleton_utility_a09b46ba8e665a6fe19196e445e10f323}{Skeleton\+Utility.\+flip\+By180\+Degree\+Rotation} to true, not via negative scale. 

Note\+: This component is automatically attached when calling "{}\+Create Hinge Chain"{} at \doxylink{class_spine_1_1_unity_1_1_skeleton_utility_bone}{Skeleton\+Utility\+Bone}, do not attempt to use this component for other purposes.
