\doxysubsection{상세한 설명}
Stores the current pose for a transform constraint. A transform constraint adjusts the world transform of the constrained bones to match that of the target bone.

See \href{http://esotericsoftware.com/spine-transform-constraints}{\texttt{ Transform constraints}} in the \doxylink{namespace_spine}{Spine} User Guide.
