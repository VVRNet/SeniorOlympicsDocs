A simple fadeout component that uses a \doxylink{class_spine_1_1_unity_1_1_examples_1_1_skeleton_render_texture}{Skeleton\+Render\+Texture} for transparency fadeout. Attach a \doxylink{class_spine_1_1_unity_1_1_examples_1_1_skeleton_render_texture}{Skeleton\+Render\+Texture} and this component to a skeleton Game\+Object and disable both components initially and keep them disabled during normal gameplay. When you need to start fadeout, enable this component. At the end of the fadeout, the event delegate {\ttfamily On\+Fadeout\+Complete} is called, to which you can bind e.\+g. a method that disables or destroys the entire Game\+Object.
